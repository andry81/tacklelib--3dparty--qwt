1. DESCRIPTION
-------------------------------------------------------------------------------
qwt patched sources fork from:
https://svn.code.sf.net/p/qwt/code

Web access:
  https://sf.net/p/qwt
  https://qwt.sourceforge.io

From authors:

  The Qwt library contains GUI Components and utility classes which are
  primarily useful for programs with a technical background.
  Beside a framework for 2D plots it provides scales, sliders, dials,
  compasses, thermometers, wheels and knobs to control or display values,
  arrays, or ranges of type double.

The original library patched to add these files:

  * changelog file
  * readme file
    Cmake scripts uses the cmake modules from the tacklelib library:
    https://svn.code.sf.net/p/tacklelib/tacklelib/trunk/cmake/tacklelib
